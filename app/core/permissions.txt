# SINAS Permission Definitions
# Format: sinas.{resource}.{action}:{scope}
#
# Actions:
#   - Standard CRUD: get, post, put, patch, delete (maps to HTTP verbs)
#   - Custom actions: execute, search, manage_members, etc. (semantic names)
#
# Scopes: own, group, all
#   - own: user's own resources
#   - group: user's group resources
#   - all: all resources (typically admin)
#
# Wildcards:
#   - sinas.*:all (admin - all permissions)
#   - sinas.*.get:all (read-only admin)
#   - sinas.agents.*:own (all operations on own agents)

# Agents - AI Agent Management
# Format: sinas.agents.{namespace}.{action}:{scope}
# Examples:
#   sinas.agents.default.post:own (create agent in default namespace)
#   sinas.agents.support.get:group (view support namespace agents)
#   sinas.agents.*.post:own (create agent in any namespace you own)
